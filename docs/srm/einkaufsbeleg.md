# Einkaufsbeleg
[⬅ Zurück zur Übersicht](../index.md)
## Einleitung
Dieses Handbuch erklärt, wie Sie im Modul **SRM** einen **Einkaufsbeleg** anlegen.  
Die Funktion dient dazu, neue Anfragen, Bestellungen oder andere Beschaffungsbelege zu erstellen und an Lieferanten zu übermitteln.  
Sie benötigen die entsprechenden **Berechtigungen im SRM-Modul** sowie aktivierte **Belegtypen**, die zu Ihren Geschäftsprozessen passen.

---

## Schritt-für-Schritt-Anleitung

### 1. Zugang
Öffnen Sie das Menü **SRM > Einkaufsbeleg anlegen**.  
Hier werden alle für Ihr Unternehmen **aktivierten Belegtypen** angezeigt.

---

### 2. SRM Belegtypen prüfen
Die angezeigten **Belegtypen** entsprechen den im System konfigurierten Geschäftsprozessen.

**Anpassungsmöglichkeiten:**
- Wenn Sie zusätzliche Belegtypen benötigen, können diese durch die Administration aktiviert werden.  
- Nicht benötigte Belegtypen lassen sich ausblenden.  

Die zugehörigen Einstellungen finden Sie unter **Admin > Properties > Belegtypen**.

---

### 3. Belegtyp auswählen
1. Klicken Sie auf den gewünschten **Belegtyp**, z. B. **Anfrage**.  
2. Nach der Auswahl werden Sie automatisch zur **Seite „Anfrage anlegen“** weitergeleitet.

---

### 4. Belegdaten eingeben
1. Füllen Sie die **Pflichtfelder** aus:  
   - **Belegdatum** (voreingestellt ist **heute**, kann angepasst werden)  
   - **Kreditor** (geben Sie den gewünschten **Lieferanten** ein)  
2. Prüfen Sie die Felder **Ersteller** und **Besitzer**.  
   - Diese sind standardmäßig mit dem aktuell angemeldeten Benutzer vorbelegt.  
   - Je nach **Systemkonfiguration** und **Benutzerrechten** können Sie sie anpassen.  
3. Ergänzen Sie bei Bedarf **optionale Felder**, z. B. **Projekt**, falls zusätzliche Informationen erforderlich sind.  
4. Klicken Sie auf **Speichern**.  
   Das System generiert automatisch eine **Belegnummer** und öffnet die Detailansicht des neu erstellten Belegs.

---

## Zusammenfassung
Mit dieser Funktion können Sie Einkaufsbelege effizient und standardisiert im Modul **SRM** anlegen.  
Achten Sie darauf, die erforderlichen Pflichtfelder korrekt auszufüllen und die passenden Belegtypen zu wählen.  
So stellen Sie sicher, dass alle Belege reibungslos verarbeitet und dem richtigen Lieferanten zugeordnet werden.
