# Lieferanten
[⬅ Zurück zur Übersicht](../index.md)
## Einleitung

Dieses Kapitel beschreibt, wie Sie im Modul **SRM** mit Lieferanten arbeiten.  
Sie erfahren, wie Sie Lieferanten suchen, über Filter eingrenzen und neue Lieferanten anlegen.  
Die beschriebenen Schritte gelten für alle Benutzer, die im Einkauf oder in der Stammdatenpflege tätig sind.  
Voraussetzung: Sie verfügen über Zugriffsrechte auf das Modul **SRM**.

---

## Schritt-für-Schritt-Anleitung

### 1. Lieferantensuche

1. Öffnen Sie **SRM > Lieferanten**.  
2. Geben Sie im Feld **Firmennamen** Stichwörter ein.  
   Das System zeigt alle Lieferanten im Vorschlagsfenster an, die die eingegebenen Stichwörter enthalten.  
   (Die Stichwörter können sich auf Anteile der Suchkriterien beziehen, z. B. Firmenname. Optional kann auch nach Stadt oder Telefonnummer gesucht werden; diese Angaben werden im Suchfeld berücksichtigt.)  
3. Je mehr Zeichen eingegeben werden, desto präziser ist das Suchergebnis.  
4. Lieferantentreffer erscheinen automatisch im Vorschlagsfenster, sobald Sie Zeichen eingeben.  
5. Wählen Sie den gewünschten Lieferanten aus.  

**Hinweis:**  
Wenn Sie einen allgemeinen Suchbegriff eingeben, zeigt das System alle Lieferanten, die die eingegebene Zeichenkette enthalten.

---

### 2. Suche nach Filterkriterien

1. Legen Sie den gewünschten Suchbegriff fest.  
2. Klicken Sie auf **Abfrage starten**.  
3. Das System zeigt alle Lieferanten an, die den gesetzten Filtern entsprechen, z. B. nach dem Filterkriterium **Anlagedatum**.  
4. Wenn Sie Lieferanten suchen, die an einem bestimmten Tag angelegt wurden, geben Sie diesen Tag im Feld **Anlagedatum** ein.  
5. Die entsprechenden Lieferanten werden in der unteren Tabelle angezeigt.

---

### 3. Lieferant anlegen

1. Klicken Sie auf **Lieferant anlegen**.  
   Das Pop-up-Fenster **Lieferant anlegen** wird geöffnet.  
2. Geben Sie im Fenster die grundlegenden Informationen für den neuen Lieferanten ein:  
   - **Lieferantentyp** (Firma oder Person)  
   - **Firmenname** (oder Vor- und Nachname bei einer Person)  
3. Beachten Sie, dass beide Felder Pflichtfelder sind.  
4. Weitere Informationen können bei Bedarf jetzt oder später ergänzt werden.  
5. Klicken Sie auf **Speichern**.  
6. Das System leitet Sie automatisch auf die Startseite des neu angelegten Lieferanten weiter.  
   Dort sehen Sie die grundlegenden Informationen und die vom System automatisch generierte **Lieferantennummer** (z. B. 168927).

---

## Zusammenfassung

Mit den beschriebenen Schritten können Sie Lieferanten effizient suchen, nach Kriterien filtern und neue Datensätze anlegen.  
So stellen Sie sicher, dass Ihr Lieferantenstamm stets aktuell und vollständig bleibt.  
Für weiterführende Informationen siehe Kapitel **Lieferantenstammdaten**.
